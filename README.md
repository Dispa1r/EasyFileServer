# EasyFileServer
## 一个用go写的简单的网盘系统，参考教程
## 做了几点优化
#### 数据全部使用gorm操作
#### 添加邮箱验证功能
#### token迁移至redis，删除token表
#### jwt采用了go-jwt
#### 支持分块上传和断点续传和秒传
#### 根据用户名隔离存储文件
